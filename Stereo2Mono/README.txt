

- Stereo2Mono - converts a given stereo file into a mono file with 3 possible options

	1. get left channel, 
	2. get right channel 
	3. Average of both channels

Note: This program relies on the libsndfile library, rather than portsf, which is available for free online and must be installed before use. If you have brew installed, you can simply do brew libsndfile.

Tip: Test the program on the provided test file to compare each method. The panning in the provided file will help highlight the audible differences between the methods.
